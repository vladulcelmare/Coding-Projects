## Numerical Methods

### Linear Ecuation Solver (`Linear_System_Solver.py`)

  A python application used for computing m x n linear ecuations, supports complex numbers and featuring :
- Performs **LU factorization** for any kind of matrices that support LU 
- Implemented with Doolittle , used as comparison -> **numpy**
- Supports random input + import .txt files

- Performs **QR factorization** for any kind of matrices that support QR
- Implemented with HouseHolder , used as comparison -> **numpy**
- Supports random input + import .txt files

- Dynamic inputs
- GUI : Implemented with **tkinter** ( familiar thanks to Total Commander project )
- detailed console log with possible errors, current operations, detailed explanations

## Usage 
- Educational purposes ( diving deeper into tkinter, applying LU and QR efficiently on complex and diverse linear systems )

### Custom MLP from Scratch (`newnn.py`)

  A fully manual implementation of a **Multi-Layer Perceptron**, inspired by how frameworks like PyTorch work internally, featuring :
- Character-level MLP
- Probability principles ( normalisation, variance, standard deviation )

- Accurate loss tracking

## Usage
